# Fee

Represents a fee and is used across all all PerfectGym API.


## Fee properties

Fee is described by the following properties


Name       | Type       | Description
-----------|------------|-----------
`net`      |`decimal`   | Net fee.
`tax`      |`decimal`   | Tax value.
`taxRate`  |`decimal`   | Tax rate.
`gross`    |`decimal`   | Gross fee.
