--- kind: change title: Sortable Stars in Repository Starring API created_at: 2013-02-14 author_name: pengwynn --- As we [announced on the GitHub blog][post], Stars now support sorting. The Repository Starring API now supports [two new parameters][params] when listing Stars: `sort` and `direction`. ``` command-line $ curl https://api.github.com/users/defunkt/starred?sort=created&direction=asc ``` Enjoy.
