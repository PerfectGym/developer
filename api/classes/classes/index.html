Classes

This API gives you read only access to list of fitness classes and classes types conducted in your gym.

  • Classes types
  • Classes
  • User classes

Classes types

Each fitness class has a class type assigned . Class type is for example 'Zumba', and a class is an instance of a class type (for example Zumba at 'Fibo' club, starting on 2015-12-21 18:00). Class types are global accross all clubs of a company (in case company has more then 1 club).

List classes in a given timeframe

GET Classes/GetClasses/{clubId}/{startDate}/{endDate}

List classes with timestamp

GET Classes/GetClasses/{clubId}/{timestamp}
